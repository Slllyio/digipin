Analytics Engine

Urban Query Engine

52 pre-built analytical queries spanning 7 urban sectors. Each query samples 25 points, scores them against 160+ features, and ranks the top locations.

  • Commercial — best retail, restaurant, office spots
  • Residential — family-friendly, affordable areas
  • Infrastructure — connectivity, transit access
  • Green — parks, tree cover, open spaces
  • Safety — well-lit, low-noise, emergency access
  • Governance — ward coverage, civic amenities

AI Assistant

Meet DISHA

DISHA is an AI assistant that answers natural language questions about any location. It runs on a local LLM (Qwen2.5 via Ollama) with the full intelligence profile injected as context.

  • Multi-provider — Ollama, Groq Cloud, or custom API
  • Smart context — sends only relevant data per question
  • Response cache — IndexedDB with TTL per data type
  • City scan — batch analyze all cells at once
  • Privacy-first — local inference, no data leaves your machine
